Jeff Walley's Week 03 & Week 04 Webpage with External CSS Stylesheet

This week we link an external CSS (Cascading Style Sheet) =>

<link rel="stylesheet" type="text/css" href="style.css"/>

The 'link' tag allows us to specify an external path that holds the webpage's CSS style file.
By keeping the CSS file in an external location we can have a cleaner HTML structure. Having an external file also helps with templating and reusability of our webcode.

The 'rel', or relation/ship, attribute is equal to "stylesheet". This tells the browser that we are using a stylesheet in the link.

Next we have the 'type' attribute, which tells the browser that the stylesheet is in "text/css" format.

Finally we have the 'href' attribute. The href is where you enter the URL of a page you want to use as link.
