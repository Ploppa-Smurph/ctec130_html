W3 Schools - HTML Attributes

    - some elements will have 'attributes' associated with them; ex: (a href) - in this example 'a' is the element and 'href' is an attribute of the element 
    - another ex: (img src) -> 'img' is the element and 'src' is an attribute of the element
    - 'width' and 'height' can be attributes of 'img' element, also
    - 'alt' attribute is a required attribute of the 'img' element
    - 'style' is actually an attribute
    - 'title' is an attribute, also, can create a mouseover text
